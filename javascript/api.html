XPathHelper: (xh)¶

class XPathHelper(currentPath)¶

exported from xpath-helper

XPathHelper provides a simple and chainnable API to build complicated XPath queries without the hassle. After building your XPath query, call the method toString() to get the corresponding XPath string.

Creates an instance of XPathHelper.

Arguments

  • currentPath (string[]()) –

XPathHelper.empty()¶

Empties the current path.
